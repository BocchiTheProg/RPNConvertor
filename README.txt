Інструкція по використанню
Для запуску програми використовуюте командну строку. Ex: ruby ConvertorRPN.rb
Вираз для перетворення вводиться як аргумент командної строки
В самому виразі розділяєйте операнди на оператори (та дужки) пробілами
Конвертор може розпізнати 4 найпростіші математичні оператори:
Додавання: +
Віднімання: -
Множення: x або "*"
Ділення: /
Для позначення операції множення використовуйте знак х або "*"
Для використання дужок у виразі використовуйте "(" та ")"
Ex: ruby ConvertorRPN.rb 24 + 5 "*" "(" 23 / 21 ")" / 2
